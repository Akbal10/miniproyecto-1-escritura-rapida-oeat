# Escritura Rápida - Abyss Trial

Typing game built with JavaFX inspired by dark fantasy aesthetics.

## Features
- Level-based difficulty system
- Dynamic countdown timer
- Themed vocabulary (Abyss / Dark Fantasy)
- Visual feedback and UI styling

## Technologies
- Java
- JavaFX
- FXML

## How to run
Run `Main.java`

## Documentation
JavaDoc available in `/docs/index.html`
